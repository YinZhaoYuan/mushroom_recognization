# Tensorflow model converter

### mnist model export example
#### Usage:
* Step[0]  
training mnist CNN model.
* Step[1]  
verify the accuracy of tensorflow model.
* Step[2]  
convert tensorflow model to easycnn model.
* Step[3]  
verify the accuracy of easycnn model.
